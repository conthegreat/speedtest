How to set this up (for conthegreat)

  1. Upload this index.html to your GitHub repo conthegreat/speedtest (or whatever repo name)
  2. Enable GitHub Pages and connect the custom domain speedtest.networkcontractor.co.uk
  3. Create three files in the repo root:
    10mb   •   100mb   •   1gb
  4. Commit and push — the clean URLs above will work immediately.
For the larger files (especially 1GB), consider using Git LFS or host via GitHub Releases and add simple redirect pages if needed.
